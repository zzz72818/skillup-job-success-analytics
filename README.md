# SkillUp Job Success Analytics

## Overview

This project analyzes workforce development and job seeker engagement data to identify key drivers of:

- Training participation
- Successful job placement

Using People Analytics and machine learning techniques, this project evaluates how user engagement, demographics, and career milestones influence employment outcomes.

---

## Business Problem

SkillUp is a nonprofit organization that helps workers without college degrees secure quality jobs through career navigation, training, and employment resources.

The goal of this project is to identify:
- What behaviors increase training participation
- What factors improve job placement success
- Which demographic and engagement variables are most influential

---

## Dataset

Data Sources:
- WAP_pulse_survey
- WAP_all_time_connect
- WAP_individual

Analysis Period:
- Q1 2023 – Q3 2024

Total Records:
- 18,696 users

---

## Methodology

### Data Cleaning
- Removed duplicate records
- Handled missing values
- Created engagement behavior variables

### Statistical Analysis
- Chi-Square Test
- Correlation Analysis

### Machine Learning Models
- Logistic Regression
- Random Forest

### Target Variables
- `skillup_training`
- `skillup_find_new_job`

---

## Key Findings

- Platform engagement strongly correlates with employment success
- Users who actively applied for jobs or training had higher success rates
- Technology and Healthcare industries showed stronger outcomes
- Demographics and regional preferences significantly impacted results

---

## Recommendations

- Implement reminder systems for inactive users
- Develop personalized career roadmaps
- Improve region-specific workforce strategies
- Expand support for high-growth industries
